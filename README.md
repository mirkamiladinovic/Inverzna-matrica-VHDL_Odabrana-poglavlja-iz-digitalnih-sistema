Inverzna matrica VHDL — Odabrana poglavlja iz digitalnih sistema
Kratak opis
Ovaj repozitorij sadrži VHDL implementaciju modula za računanje inverzne matrice (projekt ili vežba u okviru kolegija "Odabrana poglavlja iz digitalnih sistema"). Repo sadrži izvorne VHDL fajlove, testbencheve za verifikaciju i dodatnu dokumentaciju/primere.
